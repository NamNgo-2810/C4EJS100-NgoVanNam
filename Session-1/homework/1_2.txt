1.1 The cases that cause SyntaxError when declare variable in JS:
    Declare a variable twice.
    Eg: let a = 6;
	let a = 7; // SyntaxError: 'a' has already been declared
    Declare invalid name for variable.
    Eg: let 6c; // cannot start with a digit
	let my-name; // variable name cannot include special characters (except from '$' and '_')
    Using default keyword to declare variable name.
    Eg: let let = 5; // "let" is a default keyword
        let var = 5; // "var" is a default keyword
1.2 We can use typeof method to check data type of variable.
    Eg: console.log(typeof 3); // Output: "number"
        console.log(typeof 'hello') // Output: "string"
